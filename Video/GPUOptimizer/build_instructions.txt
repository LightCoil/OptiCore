# Инструкции по сборке GPU Optimizer X360

## Требования
- Xbox 360 XDK (например, через Visual Studio 2005/2008 с надстройкой Xbox)
- Совместимость с Freeboot-прошивкой и Aurora

## Команда для сборки
Используйте следующую команду для компиляции проекта в исполняемый файл .xex:

```
cl /D _XBOX /D _DEBUG /I include src/*.cpp /link /OUT:build/gpu_optimizer.xex
```

## Примечания
- Убедитесь, что пути к файлам исходного кода и заголовочным файлам указаны правильно.
- После сборки файл gpu_optimizer.xex можно запустить через файловый менеджер Aurora на консоли Xbox 360 с Freeboot-прошивкой.
